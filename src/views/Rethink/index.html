RethinkDB GUI Client for Mac OS X and Windows

Clean and Simple Interface

ReQL Pro provides a clean and simple interface for you to browse and modify your RethinkDB data quickly.

Frequently asked Questions

Why ReQL Pro? Why not use the RethinkDB web interface?

To browse data using the RethinkDB web interface you must manually type ReQL commands. ReQLPro provides a much more flexible user interface allowing you to browse, page, sort, edit, and delete, with familiar, easy to use point and click controls.

Is ReQL Pro Open Source?

Not yet, but we are looking forward to open sourcing it in the future, so stay tuned.

How do I report bugs and suggest new features?

Head over to our Help Center and send us a message. We appreciate your feedback because it helps us improve ReQLPro, so thanks!

ReQLPro: The Product, The Vision

Here's what you have to look forward to in future releases of ReQLPro.

See below for some of our initial ideas and features we would like to add. We're excited to get these features integrated, and we encourage you to provide feedback to help us prioritize.

Planned Features:
  • Advanced Searching
  • Update tree view to be consistent with edit / update
  • Add ability to traverse nested object trees in table view
  • Changefeed functionality
  • Raw ReQL command output
  • Raw ReQL command input
  • Save and display a history of recent searches / saves / deletes
  • Ability to undo deletes / updates / inserts
  • View and Update Indexes
  • Add support for Auth Keys
  • Add support for Users / Permissions
  • Detailed Query performance / Stats
Bigger Ideas:
  • MongoDB wrapper
  • Redis wrapper
  • CouchDB wrapper
  • Plug In System for extending data visualizations and query builders
