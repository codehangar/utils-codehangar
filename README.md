- src (app source code)
	- components (reusable elements)
 		- Component
 			- Component.html

 			- Component.scss
	- views (pages)
		- View
			- View.html
			- View.js
			- View.scss
	- index.html
	- index.js
	- index.scss
- tests (automated tests)
- utils (build tools)
- LICENSE
- package.json
- README.md
